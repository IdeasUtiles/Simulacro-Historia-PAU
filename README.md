# Historia PAU · Simulacro Azul (v1)
App web estática lista para GitHub Pages. No requiere servidor.

## Modos
- **Estudiar**: microresúmenes + mnemotecnias + preguntas
- **Simulacro PAU 90 min**: selección inteligente (prioriza fallos y no vistas)

## Abiertas (corrección fácil)
- Obligatorio escribir un párrafo (mínimo caracteres).
- La app puntúa por **criterios**: lista de palabras/ideas clave.
- Feedback: qué incluiste / qué te faltó + respuesta modelo.

## Deploy en GitHub Pages
1) Repo nuevo → Upload files → sube todo a la raíz
2) Settings → Pages → Deploy from branch → main / (root) → Save
3) Abre la URL que te da GitHub
